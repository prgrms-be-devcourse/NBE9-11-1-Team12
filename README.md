# NBE9-11-1-Team12

☕ 카페 메뉴 관리 서비스 (Grids & Circles)
Team12 | 2026.03.26
----------------------------------------------------------------------
📝 프로젝트 소개

우리는 작은 로컬 카페 Grids & Circles입니다.
고객은 웹사이트를 통해 커피 원두를 주문하고,
관리자는 상품과 주문을 효율적으로 관리할 수 있습니다.

카페 메뉴 데이터를 CRUD 기반으로 관리하고 프론트엔드 - 백엔드 분리 구조 설계를 하였고 사용자와 관리자 기능을 분리한 웹 서비스 입니다.

----------------------------------------------------------------------
🎯 프로젝트 목표

- REST API 기반 백엔드 설계 및 구현
 
- 프론트엔드와 백엔드의 데이터 흐름 이해
  
- Git을 활용한 협업 경험 및 브랜치 전략 학습

----------------------------------------------------------------------
🏗️ 시스템 구조

- Frontend: Next.js
  
- Backend: Spring Boot (REST API)
  
- Database: H2 (개발 환경)
  
- 협업 도구: GitHub, Issue, PR

----------------------------------------------------------------------
🧩 역할 분담

👤 사용자 기능
1. 상품 목록 조회
- 상품명, 가격 리스트 출력

- 상품 선택 및 수량 설정
  
- 주문 파라미터 전달 (productId, quantity)

2. 주문 생성
이메일, 주소, 우편번호 입력
주문 생성 요청 (POST)
총 금액 계산 및 주문 완료 처리

3. 주문 조회
이메일 기반 주문 조회
주문 상태 확인 (배송중 / 완료)
주문 취소 기능 (배송 전 가능)

🛠 관리자 기능
1. 상품 관리
상품 조회 / 등록 / 수정 / 삭제 (CRUD)
JSON 기반 데이터 처리

2. 주문 관리
전체 주문 조회
주문 상태 변경 (배송 완료 처리)

3. 주문 집계
동일 고객의 주문을 일정 시간 기준으로 묶어서 처리
(전날 14시 ~ 당일 14시)
