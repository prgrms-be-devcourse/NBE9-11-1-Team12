# NBE9-11-1-Team12

☕ 카페 메뉴 관리 서비스 (Grids & Circles)
Team12 | 2026.03.26
----------------------------------------------------------------------
📝 프로젝트 소개

우리는 작은 로컬 카페 Grids & Circles입니다.
고객은 웹사이트를 통해 커피 원두를 주문하고,
관리자는 상품과 주문을 효율적으로 관리할 수 있습니다.

카페 메뉴 데이터를 CRUD 기반으로 관리하고 프론트엔드 - 백엔드 분리 구조 설계를 하였고 사용자와 관리자 기능을 분리한 웹 서비스 입니다.

----------------------------------------------------------------------
🎯 프로젝트 목표

- REST API 기반 백엔드 설계 및 구현
 
- 프론트엔드와 백엔드의 데이터 흐름 이해
  
- Git을 활용한 협업 경험 및 브랜치 전략 학습

----------------------------------------------------------------------
🏗️ 시스템 구조

- Frontend: Next.js
  
- Backend: Spring Boot (REST API)
  
- Database: H2 (개발 환경)
  
- 협업 도구: GitHub, Issue, PR

----------------------------------------------------------------------
🧩 역할 분담

👤 사용자 기능
1. 상품 조회
- API: 상품 목록 조회, 주문 파라미터 전달 (R)
  
- Front: 상품 목록, 네비게이션(주문조회 버튼), 상품목록 마지막 주문하기 버튼
2. 주문 생성
- API: 주문생성 (C)
  
- Front: 주문 생성 폼, 구매 성공 화면

3. 주문 확인
- API: 본인 주문 조회 (이메일 입력 받고, 해당 이메일 주문 조회), 시간남으면 주문 취소 (RD)
  
- Front: 주문 조회 화면

🛠 관리자 기능
1. 상품 조회
- API: 상품 목록 조회, 상품 상세 조회 (R)
  
- Front: 관리자 상품 목록 조회, 상품 상세 조회

2. 상품 추가/수정/삭제
- API : 추가, 수정, 삭제 (CUD)
  
- Front : 추가/수정/삭제 폼

3. 주문관리(주문조회, 주문완료처리)
- API: 주문 목록 조회, 주문 완료 처리 (RU)
  
- Front: 주문 목록, 주문완료 처리 버튼
